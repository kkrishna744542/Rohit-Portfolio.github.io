# Rohit Raj - Personal Portfolio

A modern, responsive, and feature-rich personal portfolio website designed to showcase professional skills, projects, and provide a direct line of communication for potential clients and collaborators.

## 🚀 Live Demo
You can view the live project here: [https://kkrishna744542.github.io/Rohit-Portfolio.github.io/](https://kkrishna744542.github.io/Rohit-Portfolio.github.io/)

## ✨ Features

* **Dynamic Theme Switcher:** Includes 5 custom themes: Light, Dark, Ocean, Sunset, and Forest (default).
* **Animated UI:** Smooth fade-in animations and a professional typing effect in the header.
* **Interactive Contact & Enquiry Forms:** Integrated with **EmailJS** for real-time email notifications without a backend.
* **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
* **Skill Showcase:** Categorized display of technical expertise including React, Node.js, and UI/UX Design.
* **Visual Gallery:** A curated grid to highlight projects and creative work.

## 🛠️ Built With

* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Custom styling including Flexbox, CSS Grid, and CSS Variables for the theme engine.
* **JavaScript (ES6+):** Logic for the mobile menu, scroll effects, and theme switching.
* **EmailJS:** To handle form submissions directly to your inbox.
* **Font Awesome:** For professional iconography.
* **Google Fonts:** Using 'Poppins' for a clean, modern aesthetic.
